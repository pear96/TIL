# 오버로딩 vs 오버라이딩

|                          | 오버 로딩                                   | 오버 라이딩                         |
| ------------------------ | ------------------------------------------- | ----------------------------------- |
| 의도                     | 기존에 없는 **새로운** 메서드를 정의하는 것 | **상속받은 메서드를 재정의**하는 것 |
| 메서드 이름              | 동일                                        | 동일                                |
| 매개변수 개수,순서, 타입 | 개수, 순서, 타입 중 하나라도 달라야함       | 동일                                |
| 반환타입                 | 상관 없음                                   | 동일하거나 하위 타입                |
